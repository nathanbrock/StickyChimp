StickyChimp Example

StickyChimp is a jQuery plugin which mimics the sticky menu functionality found on the new MailChimp website (http://www.mailchimp.com). The menu is hidden as you start to scroll down the page and your focus moves to the content itself. If you start to scroll back to the top of the page (where the menu normally resides) or reach the bottom of the page the menu will reappear.

Using the plugin is simple, call the StickyChimp plugin on your menu element (that is position: fixed) and away you go.

Example

$('.menu-wrap').stickychimp();

Questions / Issues?

Any questions? Feel free to post them via Github at https://github.com/nbrock/StickyChimp/issues.
